I’m Justin Schomer, a self‑taught security researcher and founder of Caseit2u2 Secure Labs.

I discovered a hardcoded architectural backdoor (46D1 / 4ccb9e46) in firmware used by Rockwell Automation, Siemens, and Android Debug Bridge (ADB). The backdoor allows remote, zero-click authentication bypass and includes an explicit unlock command (action=unlock), proving intentional design.

I validated it against a live U.S. Department of Defense IP, reported it to CISA (VU#302619), and built a patent‑pending defensive framework (ETB AAV LayerZero) that blocks this entire class of vulnerability.

I did this from a non‑rooted Android phone.
